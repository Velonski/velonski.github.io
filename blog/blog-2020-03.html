  • Перенос velonski.ru на Github Pages

    19.03.2020

    Решил уменьшить расходы на хостинг сайта velonski.ru. В итоге переместил сайт на бесплатный хостинг Github Pages. Почитать об этом хостинге можно по адресу https://pages.github.com.

    Там можно держать только статичные сайты с открытым исходным кодом, но этого мне вполне достаточно. Мой личный сайт не приносит мне никакого дохода, поэтому более целесообразно содержать его без использования денег.

    Есть ограничения на объём свободного места для ресурсов сайта, поэтому надо изменить принцип наполнения: теперь буду добавлять на сайт преимущественно текстовый контент. Любые картинки, документы и прочие файлы занимают много места, их буду держать в каких-нибудь облачных хранилищах.

    Код сайта velonski.ru теперь доступен для просмотра на github.com. Ссылка на репозиторий: https://github.com/Velonski/velonski-github-notes

    У сайта был до этого другой репозиторий, который теперь я перестану обновлять. Ссылка: https://github.com/Velonski/velonski-ru-html

    Кстати, есть у меня сайт-портфолио: web.velonski.ru. Он тоже теперь "хостится" на Github Pages. Код доступен по ссылке: https://github.com/Velonski/web-velonski-ru

  • Antix Linux: удобная ОС для старого оборудования

    20.03.2020

    В последнее время я очарован операционной системой Antix Linux. Она поставляется в вариантах на 32 и на 64 бита, поддерживает старое оборудование, использует минимум ресурсов на этапе старта рабочего стола, "из коробки" предоставляет набор программ для всего самого необходимого, может полноценно работать без установки на жесткий диск, в оперативной памяти.

    Сейчас я ежедневно использую эту операционную систему на стареньком компьютере с процессором Intel Pentium 4, 1 Гб оперативной памяти и интегрированной видеокартой. Этого мне хватает, чтобы, например, пользоваться современным Интернет через браузер Firefox ESR, хотя лучше не открывать более 3 вкладок с "тяжёлыми" сайтами. Браузер - самое ресурсоёмкое приложение, всё остальное ПО работает нормально.

    Думаю, Antix Linux - самый удобный Debian-based дистрибутив Linux сегодня.

    У дистрибутива есть официальный сайт: https://antixlinux.com
