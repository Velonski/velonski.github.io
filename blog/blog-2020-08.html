  • PhotoFilte 7 - легковесная замена Photoshop и GIMP на платформе Windows

    05.08.2020

    Познакомился с этой программой давно. На платформах Windows это замена для редактора Adobe Photoshop: есть поддержка слоёв и достаточно функционала для обычного человека. Быстро стартует и занимает небольшой объём памяти.

    Насколько я понял, версия 7 - последняя версия, которая бесплатна для некоммерческого использования. После 2015 года программа развивается как полностью платная + появилась приставка "Pro" в названии.

    На всякий случай, сохранил программу у себя в хранилище файлов. Ищите её в каталоге "programs".

  • Драйвер для принтера HP Deskjet 3550 в Windows 7

    09.08.2020

    Столкнулся с проблемой установки драйвера для вышеуказанного принтера. Производитель в Интернет выложил для скачивания только драйвер Windows XP, а об установке драйвера в Windows 7 сказано, что драйвер встроен систему и автоматически скачивается через Центр обновлений.

    К сожалению, мой компьютер с Windows 7 не имеет подключения к Центру обновлений, поэтому получить драйвер таким способом не получилось. Вероятно, в недалеком будущем ни одно устройство с Windows 7 не сможет получить что-либо из Центра обновлений, т.к. Microsoft уже признало эту ОС устаревшей и отказывается от её дальнейшей поддержки.

    Найти файлы драйвера в Интернет не получилось. Поисковые системы дают ссылки только на программу-помошник от HP, которая драйверов не содержит и только пытается запустить процесс установки принтера через Центр обновлений.

    Решение проблемы нашлось в небольшом обсуждении на малоизвестном форуме: в Windows 7 уже содержится драйвер, подходящий для Deskjet 3550, только он предназначен для другой модели принтера: HP 910.

    При установке драйвера принтера надо выбрать опцию установки вручную (из конкретного каталога на компьютере), потом - опция "Выбрать из списка". В списке переходим в раздел принтеров HP, и там выбираем модель 910.

    С этим драйвером мой Deskjet 3550 без проблем выполнил чёрно-белую печать на листе A4. Вероятно, цветная печать тоже будет работать.
